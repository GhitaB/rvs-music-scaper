# :notes: rvs-music-scraper
Test web scraping on rvs.

Just for fun. :)

## :star: Usage:

```
$ git clone git@github.com:GhitaB/rvs-music-scraper.git

$ cd rvs-music-scraper

$ virtualenv .

$ ./bin/pip install zc.buildout

$ ./bin/buildout

$ ./bin/rvs_scraper
```

Then open the Songs folder... :heart_eyes:
